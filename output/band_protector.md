### Новинки 20

Цiмох 13.12.2001 09:20:
1) Summoning "Let Mortal Heroes Sing Your Fame"- сразу говорю что оценку ставить этому диску не буду, слишком много ИМХИ ;). В целом же скажу, что зря группа сделала шаг назад, надо было продолжать эксперементы со Stronghold (женский вокал и иже с ними). Музыка стала красивее, добрее даже, сразу с тоской вспоминаються дни жуткого Dol Guldur'a и злобного Minas Morgul'a, особенно там где на вокалах Silenius. С моей точки зрения то, что теперь почти всё делает Protector, сильно сказалось на однообразности материала, нет определённых хитов, все песни какие-то одинаковые. Ещё дико выбивают из коллеи фразы из фильмов вставленые в некоторые из песен, губят атмосферу на корню. Диск кстати родной, потому хоцца добавить в плюсы этому релизу оформление- душевные картинки.<BR><BR>2)Lux Occulta "The Mother and the Enemy" 10/10 - плодовит сей год на качественные блэк релизы. В пример можно сразу поставить этот диск: высококачественное исполнение, неприкрытая дезовая брутальность, джжазовые и рок'н'ролльные мотивчики и высший женский вокал на трёх трэках- вот так надо играть товарищи! Все это очень психоделично, хаотично и в меру романтично! ;) Слушайте на здоровье!<BR><BR>3) Enslaved "Monumension" 10/10 - ищо один замечательный релиз. На этот раз пендюлина летит прямиком из Скандинавии, где старые викинги-алкоголики сколотили, что-то абсолютно сумасшедшее. Не зря на этот диск были приглашены специалист по фольку и спец по психоделии из двух местных групп с труднопроизносимыми названиями. Диск намного лучше предыдущего- мелодичней, тяжелее и психоделичней! Некоторые моменты напоминают Uriah Heep, некоторые Pink Floyd, ну а некоторые явный Cannibal Corpse. Вокалист тоже постарался и кроме стандартных скримингов, радосто гроулит и выводит куплеты чистым голосом. Хотя старым фанатам этой группы новый диск вряд ли понравиться, от старого Инслэйвда там мало осталось.

What Happens in a<BR>Backup Session?<BR>Whether a backup session is started interactively or from the Data<BR>Protector Scheduler, the data flow and processes are as described in the<BR>following list. A typical backup session is diagrammed in Figure 7-1.<BR>Note that the data flow and processes described here are for a standard<BR>network backup. For data flow and processes specific to other types of<BR>backup, such as direct backup, refer to the related chapter.<BR>1. The Backup Session Manager (BSM) process is started on the Cell<BR>Manager system and controls the backup session. This process reads<BR>the backup specification for information on what to back up, and<BR>which options, media, and devices to use for the backup.<BR>2. The BSM opens the IDB and writes to the IDB information about the<BR>backup session, such as generated messages, details about the backed<BR>up data, and the devices and media that were used for the session.<BR>3. The BSM starts Media Agents (MAs) on the systems with devices<BR>configured for backup. A new Media Agent is started for each drive<BR>used in parallel. The number of Media Agents that can be started in<BR>the cell is limited by the number of licenses you have purchased and<BR>the cell configuration.<BR>4. The BSM starts Disk Agents (DAs) for each disk to be backed up in<BR>parallel. The actual number of Disk Agents started depends on the<BR>concurrency of Disk Agents configured in the backup specification.<BR>This is the number of Disk Agents that can be started to send data in<BR>parallel to a Media Agent, thus allowing a device to stream.<BR>5. Disk Agents read data from disks and send it to the Media Agents<BR>that write data to media. The BSM monitors the progress of the<BR>session and starts new Disk Agents and new Media Agents, as<BR>necessary.<BR>6. When the backup session is completed, the BSM closes the session.
